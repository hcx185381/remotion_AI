【第五部分：添加音频】
===============================================================================

用户："为什么没有声音？"

AI 解释：
- Remotion 默认不包含音频
- 需要手动添加音频文件

解决方案：
1. 从免费音乐网站下载（Pixabay Music, Bensound等）
2. 放到 public/background.mp3
3. 在组件中添加 <Audio src="/background.mp3" />

创建音频生成工具：audioGenerator.ts
- 使用 Web Audio API 生成简单音效
- 但主要用于浏览器实时播放，不用于视频渲染

【关键技术要点】
===============================================================================

Remotion 核心规则：
1. 所有动画由 useCurrentFrame() 驱动
2. 禁止使用 CSS 动画和过渡
3. 3D 场景必须使用 ThreeCanvas
4. 禁止使用 @react-three/fiber 的 useFrame()

依赖版本：
- react: 18.3.1
- @react-three/fiber: 8.16.8（降级以兼容 React 18）
- @remotion/cli: 4.0.414
- @remotion/three: 4.0.414
- remotion: 4.0.414
- three: 0.182.0

3D 相机模型细节：
- 机身：boxGeometry [2, 3.5, 0.8]，metalness=0.95
- 镜头：多层圆柱 + 球面，transmission=0.95
- 屏幕：动态发光，emissiveIntensity=0-1
- 材质：高级金属和光学玻璃效果

动画系统：
- spring()：弹性动画（缩放、旋转）
- interpolate()：线性插值（位置、颜色）
- Math.sin/cos：周期性动画（旋转、脉冲）
